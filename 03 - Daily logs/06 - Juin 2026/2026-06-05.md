## 💡 Découverte du jour

**Deux tokens PartSmart** : admin (`Account/Authenticate`) + portal user (`Account/PortalUserToken`). Le `refresh_token` est partagé et lié au compte **admin**, pas au portal user — à garder en tête pour le cache Go.

**Embed `stream.js` hors périmètre** : validé en HTML local, mais Midbec passe uniquement par le proxy Go (`/partsmart/search/*`) et le header mode Modèle.

**Recherche modèles vs pièces** : `WFW5620` matche en mode **Modèle** (PartSmart), pas en mode **Pièce** (`/pim/search` → `results: []`). La marque seule (`whirlpool`) ne suffit pas — utiliser un numéro de modèle ou une référence pièce.

**Port 8080 Windows** : double instance Go possible (terminal + processus orphelin) → `netstat` + `taskkill` avant relance.

## 🧠 Ce que j'ai appris

**Principe** : Valider l'auth d'une API tierce dans Postman avant de déboguer à travers Go — le token fonctionne ou non, sans ambiguïté.

**Principe** : Une recherche qui « ne marche pas » peut être un **mauvais mode** (modèle tapé en pièce) ou un **filtre données** (PIM vide), pas un bug réseau.
